HTML Dasar

HTML (HyperText Markup Language) digunakan untuk membuat struktur halaman web. Berikut adalah beberapa tag dasar:

  • <h1> hingga <h6> — untuk judul
  • <p> — untuk paragraf
  • <a> — untuk membuat tautan
  • <img> — untuk menampilkan gambar
  • <ul>, <ol>, <li> — untuk membuat daftar
<h1>Halo Dunia</h1>
<p>Ini paragraf pertama saya</p>
<a href="https://google.com" target="_blank">Kunjungi Google</a>

CSS Dasar

CSS (Cascading Style Sheets) digunakan untuk mendesain tampilan website. Berikut adalah beberapa properti umum:

  • color — mengatur warna teks
  • background-color — mengatur warna latar
  • font-size — mengatur ukuran teks
  • margin dan padding — untuk mengatur jarak
  • border — menambahkan garis tepi
h1 {
  color: #1d4ed8;
  font-size: 2em;
  background-color: #e0f2fe;
  padding: 10px;
}

JavaScript Dasar

JavaScript membuat website menjadi interaktif. Beberapa hal yang bisa dilakukan dengan JS:

  • Manipulasi elemen HTML
  • Menanggapi event (klik, ketik, dll)
  • Validasi form
  • Mengambil data dari API
document.querySelector('h1').style.color = 'red';
alert("Halo dunia!");
